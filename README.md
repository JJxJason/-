# 🏸 约球星球 — 羽毛球智能组局与球馆评价平台

> TRAE 开发者大赛 · 生活服务赛道参赛作品

## 📖 项目简介

**约球星球**是一个专为羽毛球爱好者打造的智能社交平台网站，解决球友"约球难、找馆难、记录难"三大痛点。用户可以智能组局、浏览球馆、预约场地、记录战绩、查看排行榜，一站式满足羽毛球运动的全流程需求。

- **产品形态**：响应式 Web 网站（PC + 移动端适配）
- **面向用户**：羽毛球运动爱好者（18-45岁为主）
- **技术栈**：HTML + CSS + JavaScript（纯前端，localStorage 持久化）

## ✨ 核心功能

### 1. 智能组局系统
- 创建球局：填写时间、地点、水平要求、人数限制等
- 一键报名：报名后人数自动+1，重复报名拦截
- 智能匹配推荐：AI 推荐算法，基于用户偏好匹配球局
- 多维度筛选：城市/区域/水平/时间/关键词搜索
- 过期球局自动清理

### 2. 球馆评价平台
- 30+ 球馆完整数据，覆盖 20 个城市
- 球馆详情：地址、价格、设施、评价、评分明细
- 场地预约：选日期/时间段/场地类型，在线预约
- 我的预约：查看历史预约，支持取消
- 用户自主添加球馆（含审核流程）

### 3. 个人数据中心
- 战绩统计：单打/双打胜率，环形图展示
- 胜率趋势：SVG 曲线图，支持 7天/30天/3个月切换
- 比赛记录：手动添加/删除，与胜率趋势数据联动
- 球友动态：发布/删除/点赞
- 成就系统：18 个成就，解锁条件各异

### 4. 创新亮点功能
- 🎯 **AI 智能匹配推荐** — 基于用户偏好推荐球局，显示匹配度
- 🏆 **球友排行榜** — 活跃度/胜率/参与次数三维度排名
- 📊 **打球热力图** — 12周运动数据可视化（类 GitHub 贡献图）
- 📋 **跨页面数据联动** — 报名/创建/预约数据全站同步

## 🛠 技术亮点

| 技术点 | 实现方式 |
|--------|----------|
| 数据持久化 | localStorage，含 try-catch 错误处理 |
| 动态背景 | Canvas 粒子动画（羽毛球元素飘浮） |
| 数据可视化 | SVG 曲线图 + 热力图网格 |
| 响应式设计 | CSS Grid + Flexbox + 媒体查询 |
| 用户体验 | 骨架屏、空状态、Toast 提示、返回顶部 |
| 代码健壮性 | DOM 空值检查、JSON.parse 容错 |
